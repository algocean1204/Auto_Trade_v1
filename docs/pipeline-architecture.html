4. 데이터 흐름 요약

InjectedSystem 구조

InjectedSystem
├── components
│   ├── broker: BrokerClientImpl    ← KIS API 브로커 인터페이스
│   │   ├── virtual_auth: KisAuth   ← 모의투자 인증 (openapivts:29443)
│   │   ├── real_auth: KisAuth      ← 실전투자 인증 (openapi:9443)
│   │   └── http: AsyncHttpClient   ← HTTP 클라이언트 (재시도 3회)
│   ├── cache: RedisCache           ← Redis 캐시 인터페이스
│   ├── db: AsyncSession            ← PostgreSQL 비동기 세션
│   └── ...기타 인프라 컴포넌트
├── features
│   ├── fx_manager: FxManager       ← 환율 관리자
│   ├── regime_detector             ← 시장 레짐 탐지
│   ├── position_monitor            ← 포지션 모니터링
│   ├── capital_guard               ← 자본 보호 모듈
│   └── ...기타 피처 모듈
└── running: bool                   ← 시스템 실행 상태

9. 스케줄러/크롤러 계층

FxScheduler

src/monitoring/schedulers/fx_scheduler.py
10분 주기 환율 갱신 백그라운드 루프이다. 3단계 폴백 체인.
함수역할INOUT
start백그라운드 루프 시작-None (asyncio.Task 생성)
stop백그라운드 루프 중지-None
_tick환율 1회 갱신-Redis fx:current, fx:history 갱신
_fetch_rate_with_fallback3단계 폴백 조회-tuple[float, str] (환율, 소스)
_try_kis_rate1차: KIS API-float | None
_try_naver_rate2차: 네이버 크롤링-float | None
_try_google_rate3차: 구글 크롤링-float | None

주기: 600초, 최대 이력: 720건, 폴백: 1350.0원
